== Description ==

This theme for WordPress takes us back to the blog. With a lot of design details, soft color scheme, beautiful typography and icons and a responsive layout this theme will look great on any device. Review Talk theme is perfect for blogging, but it is also a good solution for various websites. Live demo: http://wp-demo-theme.bestweblayout.com/review-talk/

= Features = 

* Cross-browser compatible
* Two-column design
* Responsive-layout
* Threaded comments
* Translation-ready
* Pagination
* Customizable header image
* Main colors: red, white, gray

== Installation ==

Manual installation:

1. Upload the "review-talk" folder to “/wp-content/themes/” directory.

Installation using "Add New Theme"

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New.
2. Search for "review talk".
3. Click "Install" button to open the theme's repository listing.
4. Click "Install" button.

Activation and Use

1. Activate the Theme through "Themes" menu in WordPress

== Theme Notes == 

= General = 

Theme  uses JavaScript for changing width of blog posts (makes one column layout) depending on sidebar height and for content and sidebar positioning depending on screen width.
For correct displaying of content theme requires enabled Javascript in browser

= Post Thumbnail Functionality =

For post thumbnails, we recommend using images with the width equal to 752px.

= Widgets =

Review Talk theme includes two additional widgets Contact Info and Social Icons(placed in /widgets folder). The following widgets are shown by default: recent posts, recent comments, archives, categories, search.

= Custom Header =

Review Talk theme uses index-1200.jpg from /image folder as default header image.

= Footer =

Theme footer has 3 additional areas for displaying widgets

== Frequently Asked Questions == 

= How do I set custom header image? =

You can set custom header image on your website from your Admin UI (Appearance). Please select Menu header => Choose Image => Browse, and then click 'Upload.' You can use image of any size depending on header content.

= How do I add a featured image? =

You can do this in Posts section > Add new or edit existing post > Set featured image.
